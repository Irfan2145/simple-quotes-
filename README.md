# DevQuote 🚀

A simple and modern motivational quotes app built using HTML, CSS, and JavaScript.

## ✨ Features

- Random motivational quotes
- Copy quote button
- Responsive design
- Modern glassmorphism UI
- Smooth hover effects
- Beginner-friendly project

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 📸 Preview

A clean and premium-looking quotes app designed for learning frontend basics and DOM manipulation.

## 🚀 How to Run

1. Download or clone this repository
2. Open the project folder
3. Run `index.html` in your browser
